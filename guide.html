1. 快速开始

1.1 安装 ScriptCat 扩展

ScriptCat 官网 → 下载对应你浏览器的扩展并安装。

1.2 设置 ScriptCat 扩展

  1. 以 edge 为例, 点击右上角的扩展
  2. 在下拉菜单中找到 「ScriptCat」, 点击右侧的 更多操作
  3. 点击管理扩展, 进入到管理页面
  4. 点击勾选 「允许用户脚本」
  5. 其他浏览器具体可参考 ScriptCat 官网 配置

1.3 安装答题脚本

  1. 打开 ScriptCat 管理面板
  2. 从「新建脚本」下拉菜单下找到「链接导入」
  3. 填入脚本地址（点击即复制）：
    https://cdn.jsdelivr.net/gh/Netsysn/userscripts/chaoxing.user.js
  4. 点击确定

1.4 填写密钥

  1. 打开超星学习通任意课程章节页面
  2. 点击页面右侧悬浮的脚本面板
  3. 切换到「答题」标签
  4. 在密钥输入框中粘贴你的 32 位密钥
  5. 刷新页面

2. 功能说明

2.1 自动检测与答题

进入章节学习页面后，脚本自动识别题目并调用 AI 作答。日志面板会实时显示进度。

2.2 自动提交

默认关闭，需在设置中手动开启。开启后，正确率达到阈值（默认 85%）自动提交。

2.3 手动模式

在设置中关闭「自动检测题目」后，进入章节页面右下角会出现「开始答题」按钮，点击手动触发。

2.4 自动过任务点

脚本会自动播放视频/音频、翻页 PPT/PDF/电子书，完成后自动跳转下一章节。

4. 常见问题

Q: 密钥在哪填？
脚本悬浮窗 → 答题标签 → 密钥输入框。填完刷新页面。
Q: 提示「Token 验证失败」？
密钥填错了或已过期。检查是否复制完整（32 位，纯字母数字），如已过期请重新购买。
Q: 答案不自动填充？
检查脚本面板是否开启、密钥是否正确。仍不行就刷新页面或换 Chrome 浏览器。
Q: 密钥过期了怎么办？
前往 发卡站 重新购买新密钥。
Q: 支持哪些平台？
超星学习通（chaoxing.com / .edu.cn 等）、智慧树（zhihuishu.com）。
Q: 用了会被检测吗？
脚本仅供学习研究，考试等重要场合请自行答题。
